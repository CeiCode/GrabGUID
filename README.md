# GrabGUID

**GrabGUID** is an addon for World of Warcraft that helps developers and players retrieve the GUID of their target player or creature.

## Features
- Retrieves the GUID of the target player or creature.
- Allows you to input an offline player's name to look up their GUID.
- Displays the GUID in a movable frame.
- Slash commands `/grabguid` or `/gg` to toggle the frame.

## Installation
1. Download and extract the **GrabGUID** folder into `World of Warcraft\_classic_\Interface\AddOns\`.
2. Restart WoW and enable the addon in the **AddOns** menu.

## Usage
- Type `/grabguid` or `/gg` to open the GrabGUID frame.
- You can input a player's name in the text box to look up their GUID.
- If you have a target selected, the addon will grab the GUID of the target.
- Displays the GUID of the selected target or the player you input in the frame.
